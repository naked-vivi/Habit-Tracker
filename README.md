# Habit Tracker

A simple weekly habit tracker built with React, TypeScript, Vite, Tailwind CSS, and date-fns.

## Features

- Add habits with trimmed names
- Track completions across the current and previous weeks
- See daily and weekly progress at a glance
- Keep habits saved in browser local storage
- Toggle individual completion days with accessible controls

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run linting:

```bash
npm run lint
```

## Project Structure

```text
src/
  components/    Reusable UI and habit list components
  context/       Habit state provider and hook
  hooks/         Browser storage helpers
```
